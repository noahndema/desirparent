# DésirParent

La première plateforme francophone de coparentalité — connecte les adultes qui veulent devenir parents ensemble.

## Stack

- Node.js + Express.js
- PostgreSQL (Neon)
- Static HTML/CSS/JS frontend

## Pages

- `/` — Landing page
- `/inscription` — Registration (3 étapes)
- `/connexion` — Connexion
- `/decouvrir` — Découvrir les profils
- `/recherche` — Recherche avancée
- `/profil` — Mon profil
- `/notifications` — Notifications
- `/messages` — Messagerie

## Local Development

```bash
npm install
npm run dev
```

## Deployment

Configuré pour déploiement Render via `render.yaml`.
